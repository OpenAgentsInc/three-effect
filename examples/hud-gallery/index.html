OpenAgents // HUD Kit

Arwes-style sci-fi HUD primitives ported into @openagentsinc/three-effect: frames (corners, lines, underline), status lights, threshold meters, dot-grid / grid-line / scanline surfaces, a pointer illuminator, separators, a pure enter/exit animator, and crisp 3D HUD text. Move the pointer to see the illuminator. HUD H2 (#5500).
